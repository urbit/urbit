ketpam, `^&`, %ktpm
============================

Convert `%gold` to `%zinc`

`^&` is a natural rune that converts a `%gold` core into a `%zinc` core.
At present, `%zinc` cores are not commonly used.

See also
--------

ketbar, `^|`, %ktbr
============================

Produces
--------

Twig: `[%ktpm p=twig]`

Sample
------

`p` is a [twig]().

Tall form
---------

    ^&  p

Wide form
---------

    ^&(p)

Irregular form
--------------

None

Examples
--------

    ~zod/try=> |=(@ 1)
    <1.gcq [@  @n <250.yur 41.wda 374.hzt 100.kzl 1.ypj %164>]>
    ~zod/try=> ^&(|=(@ 1))
    <1&gcq [@ @n <250.yur 41.wda 374.hzt 100.kzl 1.ypj %164>]>
